Attention !

Désactiver l'authentification à deux facteurs réduira significativement la sécurité de votre compte. Cette protection supplémentaire est fortement recommandée pour protéger vos données sensibles.

Si vous désactivez l'authentification à deux facteurs, vous n'aurez plus besoin de fournir un code lors de votre connexion. Toutefois, cela rendra votre compte plus vulnérable aux tentatives d'accès non autorisées.

Important : Si vous perdez l'accès à votre appareil d'authentification, utilisez plutôt les codes de secours pour vous connecter, au lieu de désactiver complètement l'A2F.
